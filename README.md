# 🚀 CryptoShot - La Passerelle Crypto de l'Afrique

![CryptoShot](https://www.genspark.ai/api/files/s/urE6N0Rl)

## 📋 Description du Projet

CryptoShot est une plateforme numérique moderne et complète permettant aux utilisateurs en Afrique d'acheter, vendre, échanger et utiliser des cryptomonnaies facilement à partir de leur Mobile Money.

Le projet vise à démocratiser l'accès aux cryptomonnaies et aux opportunités financières numériques, même avec un petit capital de départ (dépôt minimum : **2000 CFA**).

## 📖 Documentation Technique

### Structure des Vues
Chaque vue est un `<div>` avec :
- ID unique (ex: `id="deposit"`)
- Classe `view` pour le système de navigation
- Classe `active` pour la vue affichée
- Animation `fadeIn` au chargement

### Système de Navigation
```javascript
function showView(viewId) {
    // Masque toutes les vues
    document.querySelectorAll('.view').forEach(view => {
        view.classList.remove('active');
    });
    // Affiche la vue demandée
    document.getElementById(viewId).classList.add('active');
    // Scroll en haut
    window.scrollTo({ top: 0, behavior: 'smooth' });
}
```

### Gestion des États
- Pas de state management complexe
- Utilisation de `localStorage` possible pour les données persistantes
- Données temporaires en variables JavaScript
